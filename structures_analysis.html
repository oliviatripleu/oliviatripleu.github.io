# Data Structure Time Complexity
## AVL Tree:
- Insert
  - O(log n)
- Delete:
  - O(log n)
- Print in order:
  - O(n)
## Heap:
- Insert
  - O(log n)
- Delete:
  - O(log n)
- Print in order:
  - O(n log n)
  - You have to perform a deletion for every item in the heap
## Ordered Linked List:
- Insert
  - O(n)
- Finding the correct position to insert
- Delete:
  - O(1)
- Print in order:
  - O(n)
## Ordered Array:
- Insert
  - O(n)
- shifting down the items
- Delete:
  - O(1)
- Print in order:
  - O(n)
